# 📚 Atividades Práticas em Java

Este repositório contém exercícios práticos desenvolvidos em **Java**, com foco no aprendizado de lógica de programação, estruturas condicionais e interação com o usuário.

## 🚀 Objetivo

Praticar conceitos fundamentais da linguagem Java, como:

* Entrada de dados com `Scanner`
* Estruturas condicionais (`if`, `else`)
* Tipos primitivos
* Organização de código

---

## 📂 Projetos

### 🔹 1. Alistamento.java

Programa que simula um processo de alistamento baseado no peso do usuário.

**Funcionalidades:**

* Solicita o nome do usuário
* Solicita o peso
* Classifica o usuário em:

  * Abaixo do peso ideal
  * Dentro do peso ideal
  * Acima do peso ideal

---

### 🔹 2. ProjetoRenascimento.java

Programa que avalia se uma pessoa está preparada para enfrentar desafios com base em coragem e saúde física.

**Funcionalidades:**

* Verifica se o usuário possui coragem
* Avalia o nível de saúde física
* Exibe uma mensagem de aprovação ou recomendação

---

## 🛠️ Tecnologias utilizadas

* Java
* IDE (Eclipse, IntelliJ ou similar)

---

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Acesse a pasta do projeto:

```bash
cd seu-repositorio
```

3. Compile os arquivos:

```bash
javac Alistamento.java
javac ProjetoRenascimento.java
```

4. Execute:

```bash
java Alistamento
java ProjetoRenascimento
```
